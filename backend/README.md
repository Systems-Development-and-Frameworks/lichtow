# Backend

## Project setup

```
npm install
```

Create a `.env` file in this directory.

```
JWT_SECRET=<Your Secret>
process.env.NEO4J_URL=<Neo4J URL>
process.env.NEO4J_USER=<Neo4J User>
process.env.NEO4J_PASSWORD=<Neo4J Password>
```

### Run the server

```
npm run serve
```

### Lints and fixes files

```
npm run lint
```
